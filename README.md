# Java Socket Chat App

A real-time chat application with multiple clients that was constructed with Java Sockets and wing GUI.  
This project showcases multithreading, client-server architecture, and fundamental networking in Java.

# Features

Multi-client chat assistance
Broadcasting messages in real time
A GUI client that is based on Swing
Messages with a timestamp
Notifications for joining or leaving
Thread-safe server management
- Clear division of client logic, server, and GUI

# Tech Stack 

Java
Java Sockets
Swing (GUI)
The ability to multithread
